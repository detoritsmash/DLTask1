🚀 Getting Started
Prerequisites
Python 3.x

NumPy (Matrix operations)

Matplotlib (Visualization)

Scikit-Learn (Used only for initial raw data fetching)

Installation

Clone the repository:
Bash
git clone https://github.com/detoritsmash/DLTask1.git

Install dependencies:
Bash
pip install numpy matplotlib scikit-learn

Execution Options
You can run this project in two ways:

1. Using the Python Script:
Run the implementation directly from your terminal:
Bash
python deeplearningTask1.py

2. Using the Jupyter Notebook:
Open deeplearningTask1.ipynb in Google Colab or Jupyter Lab to view the step-by-step execution and interactive plots.

📂 File Structure

deeplearningTask1.py: The core NumPy implementation for the MLP and Autoencoder.

deeplearningTask1.ipynb: Interactive notebook version with detailed visualizations.

README.md: Project documentation and results.
